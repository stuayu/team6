☆サーバーを構築する

xampp：
1.データベースの作成、管理のためにxamppをインストールしてください。MySQLがすでにインストールされている場合、動かないことがあります。実行中のMySQLはタスクマネージャで落としておいてください。
2.インストールを行ったら、xampp-control.exeを開き、ApacheとMySQLのstartボタンをクリックし、MySQLのAdminボタンをクリックしてください。
3.team6という名前のデータベースを作成し、そのデータベースに本フォルダにあるteam6.sqlをインポートしてください。インポートはファイルの選択を行うだけでいいです。
4.本システムでサーバとデータのやりとりを行う場合は、ApacheとMySQLを動かしておいてください。

pipのインストール：
コマンドラインで"pip install MySQL"と入力してください。
コマンドラインで"pip install bottle"と入力してください。
コマンドラインで"pip install mysqlclient"と入力してください。
コマンドラインで"pip install PyMySQL"と入力してください。
コマンドラインで"pip install mysql-connector-python"と入力してください。
